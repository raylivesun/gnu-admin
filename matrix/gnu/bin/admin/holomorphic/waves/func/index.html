B. A model space of VCS wave functions

A model space for a compact Lie group G is a representation of G which is a direct sum of irreps comprising precisely one copy from each equivalence class of irreps [30]. As emphasised by Biedenharn and Flath [31], a model space provides a valuable framework for a realisation of the tensor algebra of the group.
